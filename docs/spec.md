# こころの羅針盤 仕様

## サービス説明

こころの羅針盤は、心を整理するための AI 相談サービスです。  
AI の名前は「ソラ」。  
ユーザーに答えを断定的に与えるのではなく、悩み・感情・問題の核・今必要なことを静かに整理する相談体験を提供します。

## ページ構成

- `/`
  - サービス紹介 LP
  - サービスの概要
  - 利用上の注意
  - 相談画面への導線
- `/consultation`
  - 相談入力
  - ソラとの対話
  - 心の整理結果
  - マイログへの導線
  - フィードバック導線
- `/log`
  - 人生の地図
  - 直近の気づき
  - 心の状態の簡易推移
  - 相談履歴一覧

## MVP機能

- 相談テーマの選択
- 自由入力による相談開始
- ソラとの 1 から 2 ターンの対話
- 心の整理結果の表示
  - テーマ
  - 感情
  - 問題の核
  - 今必要なこと
  - ソラからのメッセージ
- `localStorage` による履歴保存
- 初回サンプルデータ

## β機能

- LP の追加
- β版表示
- 利用上の注意表示
- OpenAI API を使った本実装
- フィードバック導線
- 簡易 analytics
- 心の状態メーター
- 今日の小さな気づき
- 未来のあなたからの言葉
- 次の問い
- 最近の傾向表示
- マイログ画面
- 人生の地図
